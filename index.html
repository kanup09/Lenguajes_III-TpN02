El Mundo de los Hámsters

Explora la diversidad y los cuidados esenciales para estos pequeños roedores.

Detalles del Trabajo Práctico

Tecnologías Usadas

Este proyecto implementa Bootstrap 5.3, CSS3 personalizado y una estructura semántica de HTML5 para garantizar accesibilidad.

Componentes Dinámicos

Se utiliza un carrusel dinámico con controles táctiles y transiciones automáticas cada 5 segundos mediante data-bs-ride="carousel".

Objetivo Académico

Desarrollado para la cátedra de Lenguajes III, enfocándose en el diseño responsivo y la integración de frameworks externos.
